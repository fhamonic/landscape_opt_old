Detailed Description

template<typename GR = ListDigraph, typename LEN = typename GR::template ArcMap<int>, typename TR = DijkstraDefaultTraits<GR,LEN>>
class lemon::SimplerDijkstra< GR, LEN, TR >

A minimalist Dijkstra algorithm class based on Dijkstra.

Template Parameters
GRThe type of the digraph the algorithm runs on.
LENreadable arc map that specifies the lengths of the arcs.
TRThe traits class that defines various types used by the algorithm. By default, it is DijkstraDefaultTraits<GR, LEN>
